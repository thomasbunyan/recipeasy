About

Recipeasy is a web application which aims to provide users with a unique cooking experience. It was made as part of my final year project.

Features

The application allows for creation, saving and discovery of recipes. Management by adding recipes to cookbooks; a unique method to create lists of recipes as well as a method to recommend to users recipes which might be of interest to them.

Privacy and terms

The application makes use of cookies. Therefore by making an account you consent to the use of these session tracking cookies.
